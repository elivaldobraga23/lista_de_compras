# 🛒 Lista de Compras

Aplicação web interativa para gerenciamento de lista de compras, desenvolvida com **HTML, CSS e JavaScript puro (ES Modules)**.

O layout foi baseado no design do Figma:
👉 [https://www.figma.com/design/uLGKNndS1rKLKltMjyAUwa/Lista-de-Compras-%E2%80%A2-Desafio-34--Community-?m=draw](https://www.figma.com/design/uLGKNndS1rKLKltMjyAUwa/Lista-de-Compras-%E2%80%A2-Desafio-34--Community-?m=draw)

---

## 🚀 Funcionalidades

* ✅ Adicionar itens à lista
* ✅ Validação de formulário com mensagens dinâmicas
* ✅ Seleção de quantidade e unidade (Un, Kg, Litro)
* ✅ Categorias personalizadas com cores e ícones
* ✅ Marcar item como concluído
* ✅ Exclusão com confirmação
* ✅ Feedback visual com Toastify
* ✅ Persistência de dados com LocalStorage
* ✅ Interface responsiva (Mobile First)

---

## 🧠 Conceitos Aplicados

* Manipulação avançada do DOM
* Criação dinâmica de elementos com `createElement`
* Uso de `dataset`
* Uso de `selectedOptions`
* Organização do código em módulos ES6
* Controle de estado com array de objetos
* Persistência com `localStorage`
* Controle de IDs dinâmicos
* Delegação de eventos
* Responsividade com Media Queries

---

## 🛠 Tecnologias Utilizadas

* HTML5
* CSS3 (Mobile First)
* JavaScript (ES Modules)
* Toastify.js
* Google Fonts
* Material Symbols

## 💾 Persistência de Dados

Os itens adicionados são armazenados no **LocalStorage**, permitindo que a lista permaneça salva mesmo após atualizar a página.
